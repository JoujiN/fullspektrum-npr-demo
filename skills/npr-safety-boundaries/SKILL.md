# NPR Safety Boundaries

Use this skill for all NPR onboarding and profile generation.

## Boundaries

- Do not diagnose or imply diagnosis.
- Do not say the user "has" ADHD, autism, dyslexia, anxiety, trauma, or any condition.
- Encode observed patterns as preferences, context, goals, constraints, or support needs.
- Avoid clinical, judgemental, or deterministic language.
- Let the user decline, correct, or pause without penalty.

## Sensitive Answers

If the user shares distress, trauma, health, disability, or access needs:

- acknowledge plainly and kindly
- ask only what is needed for personalisation
- keep the next question practical and optional
- preserve the user's wording where possible
